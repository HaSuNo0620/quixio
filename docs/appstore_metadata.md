# App Store メタデータ — Fiveon

## アプリ名
Fiveon

## サブタイトル（最大30文字）
5つ並べて勝利を狙え

## 説明文（最大4000文字）

Fiveon は 5×5 のボードで遊ぶ新感覚の2人用ボードゲームです。

【遊び方】
盤面の外周マスにコマを置き、押し込んで列を動かしましょう。縦・横・斜めに5つ並べた方が勝ちです。相手のコマを押し出し、自分のコマを並べる戦略が鍵となります。

【特徴】
■ AI 対戦モード
3段階の難易度（かんたん / ふつう / むずかしい）から選んで AI と対戦。ミニマックスアルゴリズムで動く手強い相手に挑戦しましょう。

■ 2人対戦モード
同じ端末で友達や家族と対戦。順番にコマを押し込んで頭脳戦を楽しもう。

■ シンプルで美しい UI
ライト/ダークテーマに自動対応。視認性の高いカラーデザインで、いつでも快適に遊べます。

■ 触り心地にこだわった演出
コマ選択・スライド・勝利のそれぞれに異なる触覚フィードバック（Haptics）と爽快なアニメーションを用意。紙吹雪エフェクトで勝利の瞬間を祝います。

■ はじめての方も安心
初回起動時にチュートリアルが表示されるので、ルールをすぐに把握できます。

シンプルなルールの中に深い駆け引きが潜む Fiveon を、ぜひ遊んでみてください。
